# Quality Assurance Engineer - Technical Test

## Running Tests

```
git clone https://github.com/Jerry1811/qa_tech_test.git
cd qa_tech_task
npm install
```

### Run UI Automation tests and generate report

```
npm run test:report
```

### Run UI Automation without report

```
npm run cypress:run
```

### Run API Automation tests

```
npm run api:test
```
